Demo Docencia Híbrida

Docencia híbrida permite gestionar sesiones de clase en las que los usuarios bien acuden directamente al aula o bien asisten a través de videoconferencia.

Se busca disponer de una actividad integrada que sea independiente del sistema de videoconferencias o de almacenamiento de vídeos de cada Universidad o institución. Profesores y estudiantes acceden a la actividad desde Moodle. Queda registrada la asistencia (ya sea en el aula o en remoto) y posteriormente se puede acceder a la grabación u otros materiales usados durante la sesión de clase. Y todo ello sin que el profesor tenga que configurar sistemas de videoconferencia o almacenamiento de videos. 

Demo KUET

Kuet es un plugin para Moodle.
Kuet le permite crear y reproducir cuestionarios interactivos para diversos propósitos y audiencias, utilizando el poder y la flexibilidad de Moodle.
Se busca disponer en Moodle de una actividad que permita organizar sesiones interactivas y atractivas de votación, respuestas rápidas y competiciones...

Demo Course Transfer

Course Transfer es un plugin de administración que permite traspasar cursos entre instalaciones Moodle de una organización.

Demo Certifygen

El proyecto de certificados amplía las funcionalidades de certificación de resultados para integrarlo con los sistemas de administración electrónica de las instituciones.

Demo Agente de notificaciones

El plugin de Agente de Notificaciones para Moodle, permite a los profesores configurar en sus asignaturas una serie de acciones automáticas para enviar mensajes de seguimiento a sus estudiantes.
Gracias a esto, los estudiantes pueden activar un conjunto de reglas para ser avisados específicamente cuando ocurran eventos específicos.
Tanto Administradores como profesores podrán configurar una serie de reglas para que los alumnos sean avisados cuando se den ciertas condiciones, excepciones y/o acciones. 

Demo Mejoras de QUIZ

Quizz es una combinación de tres plugins diseñados para ampliar la funcionalidad del quiz original de Moodle.

Este conjunto de plugins permite una gestión más avanzada del tiempo durante las evaluaciones en línea.

Además, mejora la integración Safe Exam Browser (SEB).

Demo Cloze Script

El plugin de Cloze Script para Moodle, el sistema de gestión del aprendizaje (LMS) de código abierto más popular del mundo, es una tecnología avanzada para la creación de preguntas de tipo "Cloze Calculado". Utiliza scripts sencillos para definir exámenes completos con soluciones aleatorizadas, permitiendo una personalización profunda y precisa de las preguntas de examen. Este tipo de pregunta permite la codificación de respuestas válidas a través de bloques de scripting, que evalúan la entrada del usuario basándose en lógica predefinida.

Demo Gradebook Tools

Herramientas añadidas al libro de calificaciones de Moodle para facilitar su uso a los profesores.
Contiene un asistente para generar patrones de evaluación clásicos y herramientas para obtener listados de notas más útiles.

Demo Localmail 2.0

Una completa refactorización del popular plugin "mail".
Se ha actualizado la tecnología Javascript del módulo y se han mejorado multitud de detalles funcionales.
