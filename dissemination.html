Hybrid Teaching Demo

Hybrid teaching allows you to manage class sessions in which users either come directly to the classroom or attend via videoconference.

Demo KUET

Kuet is a plugin for Moodle. Kuet allows you to create and play interactive quizzes for various purposes and audiences, using the power and flexibility of Moodle. The aim is to provide Moodle with an activity that allows you to organise interactive and engaging voting sessions, quick answers and competitions...

Project website

Demo Course Transfer

Course Transfer is an administration plugin that allows courses to be transferred between Moodle installations within an organisation.

Demo Certifygen

The certificate project extends the functionalities of performance certification to integrate with the institutions' eGovernment systems.

Demo Mejoras de QUIZ

Quizz is a combination of three plugins designed to extend the functionality of the original Moodle quiz. This set of plugins allows for more advanced time management during online assessments. In addition, it enhances the Safe Exam Browser (SEB) integration.

Demo Cloze Script

The Cloze Script plugin for Moodle, the world's most popular open source learning management system (LMS), is an advanced technology for creating "Cloze Calculated" questions. It uses simple scripts to define complete exams with randomised solutions, allowing deep and precise customisation of exam questions. This question type allows the coding of valid answers through scripting blocks, which evaluate user input based on predefined logic.

Demo Gradebook Tools

Tools added to the Moodle gradebook to make it easier for teachers to use. It contains a wizard to generate classic grading patterns and tools for more useful grade listings.

Demo Local Mail 2.0

A complete refactoring of the popular "Mail" plugin. The Javascript technology of the module has been updated and many functional details have been improved.
